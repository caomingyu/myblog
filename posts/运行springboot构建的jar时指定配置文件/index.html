运行spring boot打的jar包时，怎么指定配置文件

Written by CaoMy with ♥ on July 31, 2019 in spring boot shell jar

运行spring boot打的jar包时，怎么指定配置文件

  • 在pom.xml文件中，排除要单独指定的配置文件
          <resources>
              <resource>
                  <directory>src\main\resources</directory> <!--配置文件路径-->
                  <excludes>
                      <exclude>logback-spring.xml</exclude> <!--要排除的配置文件名-->
                      <exclude>application.yml</exclude>
                      <exclude>application-pro.yml</exclude>
                  </excludes>
              </resource>
          </resources>
  • 通过Maven打包后，执行jar包时，指定所需配置文件
  java -jar <jar包名> --spring.config.location=/home/cmy/test/ --logging.config=logback-spring.xml &
  • 进阶，配合nohup命令，实现无控制台日志输出的后台运行jar包
  nohup java -jar <jar包名> --spring.config.location=/home/cmy/test/ --logging.config=logback-spring.xml >/dev/null 2>&1&

  • 其中，如下命令实现了nohup命令无控制台日志输出功能

    nohup <...> >/dev/null 2>&1&
